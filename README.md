# Malik Akmal Naseer - Portfolio Website

A professional multi-page portfolio website showcasing my skills, projects, and experience as a Senior Backend Developer.

## Features

- **Responsive Design**: Mobile-first approach with Bootstrap 5
- **Multi-page Layout**: Separate pages for different sections
- **Modern UI**: Clean, professional design with smooth animations
- **Contact Form**: Interactive contact form with validation
- **Project Showcase**: Detailed project cards with live links
- **Skills Overview**: Comprehensive technical skills display
- **Professional Experience**: Detailed work history and achievements

## Pages

1. **Home (index.html)**: Hero section, quick stats, and featured projects
2. **Skills (skills.html)**: Comprehensive technical skills and competencies
3. **Projects (projects.html)**: Detailed project showcase with descriptions
4. **About (about.html)**: Personal information, experience, and education
5. **Contact (contact.html)**: Contact form and information

## Technologies Used
